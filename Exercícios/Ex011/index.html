Para criar ancoras de links, utiliza-se a tag (a), com o atributo (href), para criar uma referência oa hypertexto. Dentro desse atributo colocamos a URL do site

Atributos para redirecionar a pagina

Use o atributo target e seus valores

_blank

Ao usar esse valor o link vai abrir em uma nova janela

_self

Ao usar esse valor o link vai ser aberto na janela / frame atual, ele é o padrão

_top

Ao usar esse valor o link ele vai desfazer todos os frames e abrir no navegador completo

Links de downloads

Para cirar um link que faça download, basta usar o atributo "download" onde o valor será o nome do arquivo, o atributo "type" onde o valor será o tipo de arquivo, como pdf, png, jpg e etc.
